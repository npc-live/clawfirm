# ClawFirm

> Dev tool manager for one-person companies — install, manage, and orchestrate your AI-powered toolkit from one CLI.

## Quick Start

```bash
# Install globally (requires Node 18+)
npm install -g @harness.farm/clawfirm

# Log in to your account
clawfirm login

# Install all registered tools
clawfirm install

# Create a project from natural language
clawfirm new "a weather prediction trading engine"

# List registered tools
clawfirm list
```

## Commands

| Command | Description |
|---|---|
| `clawfirm login` | Log in to clawfirm.dev |
| `clawfirm whoami` | Show current session |
| `clawfirm logout` | Log out |
| `clawfirm new "<description>"` | Start a project from natural language |
| `clawfirm install [tool]` | Install all tools (or a specific one) |
| `clawfirm uninstall [tool]` | Uninstall all tools (or a specific one) |
| `clawfirm list` | List registered tools |
| `clawfirm <name> [args]` | Dispatch to clawfirm-\<name\> plugin |
| `clawfirm help` | Show help message |

## Tool Registry

| Tool | Description |
|---|---|
| **openvault** | Encrypted local secret manager |
| **skillctl** | Sync AI skills across coding tools |
| **whipflow** | Deterministic AI workflow runner |
| **agent-browser** | Browser automation for AI agents |
