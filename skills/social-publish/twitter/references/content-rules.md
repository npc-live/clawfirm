# Twitter/X 平台规则

## 平台政策要点

### 禁止行为

1. **Spam**：重复内容、批量发送、自动化垃圾信息
2. **操纵平台**：刷粉、刷赞、互粉互赞群组
3. **虚假信息**：故意传播错误信息
4. **侵权**：版权侵犯、商标滥用
5. **仇恨言论**：基于种族/性别/宗教等的歧视

### 自动化规则

```
允许的自动化:
  - 定时发推（使用官方API或授权工具）
  - 自动回复特定关键词（需透明声明）
  - RSS to Tweet（合理频率）

禁止的自动化:
  - 批量关注/取关
  - 自动点赞/RT
  - 重复发送相同内容
  - 自动回复所有mention
  - 批量DM
```

## Spam 判定条件

### 内容层面

```
触发 Spam 标记:
  - 相同或极相似内容重复发布
  - 每条推文都带链接
  - hashtag 过多（>5个/推）
  - @mention 过多（大量@不相关账号）
  - 推文中只有链接，无其他内容

安全做法:
  - 每条推文独特
  - 链接推文 < 30% 的总发推量
  - hashtag ≤ 3个
  - @ 只 mention 相关人
```

### 行为层面

```
触发限制:
  - 短时间大量关注（>100/天）
  - 短时间大量取关
  - 关注/粉丝比异常（关注远多于粉丝）
  - 短时间发送大量DM
  - 频繁创建Lists

安全阈值:
  - 关注: ≤50/天（新账号更低）
  - 取关: ≤30/天
  - 发推: ≤50/天
  - DM: ≤20/天
```

## 内容降权因素

```
确认降权:
  - 含外部链接（尤其是竞品平台链接）
  - hashtag 过多
  - 被多人举报
  - 被标记为 spam

可能降权:
  - 长期只发不互动
  - 内容与账号标签不符
  - 短时间内删除大量推文
  - 使用第三方工具的某些功能
```

### 链接降权处理

```
问题: Twitter 对外部链接推文的展示量明显降低

解决方案:
  1. 把链接放在第一条评论里（而非正文）
  2. 先发纯文字推文，再自评论补链接
  3. 用截图/图片代替链接
  4. Thread最后一条放链接（不影响首推展示）
```

## 敏感内容标记

```
需要标记为敏感的:
  - 暴力内容（即使是新闻报道）
  - 成人内容
  - 图形化医学内容

不标记的后果:
  - 推文被隐藏
  - 账号被限制
  - 严重者被暂停

如何标记:
  Settings → Privacy and Safety → Mark media as sensitive
```

## Community Notes（社区注释）

```
触发条件:
  - 内容含误导性信息
  - 数据/引用不准确
  - 缺少重要上下文

影响:
  - 推文下方显示注释
  - 大幅降低推文的传播
  - 影响账号信誉

预防:
  - 引用数据时标注来源
  - 不夸大/歪曲事实
  - 提供完整上下文
  - 区分观点和事实
```

## 账号安全

### 避免被锁定

```
常见锁定原因:
  - 短时间内大量操作
  - 从新IP/设备登录
  - 被多人举报
  - 违反自动化规则

解锁方法:
  - 手机验证
  - 等待冷却期
  - 申诉（如误判）
```

### 永久封禁风险

```
高风险行为:
  - 购买粉丝
  - 使用未授权的第三方工具
  - 规避先前的暂停
  - 严重违反平台规则
  - 操纵趋势/话题
```
