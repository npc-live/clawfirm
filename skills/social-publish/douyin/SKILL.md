# 抖音内容创作

## 文案风格

### 核心调性：口语化、信息密度高、节奏快

- **口语化**：像跟朋友说话，不要播音腔
- **前 3 秒定生死**：开场必须有强 Hook
- **信息密度**：每秒都要有信息量，不能"注水"
- **节奏感**：用语速变化、停顿、重音控制节奏
- **悬念感**：每个段落结尾留悬念，防止划走

### 口播模板

```
Hook（0-3s）:
  "[痛点/成果/反常识] + 今天就教你[解决方案]"

正文（3s-50s）:
  "第一步..., 第二步..., 第三步..."
  或 "首先..., 但关键是..., 最重要的是..."

CTA（最后5s）:
  "关注我，下期教你[下期预告]"
  "觉得有用就点赞收藏，分享给需要的朋友"
```

### 文案节奏技巧

```
短句: 用3-5字短句增加节奏感
  "注意。这很重要。记住了。"

反转: 先说常识，再推翻
  "你以为...？其实不是。真相是..."

数字具象化: 用具体数字代替模糊描述
  "3个月" > "一段时间"
  "涨了2万粉" > "涨了很多粉"

重复强调: 关键信息重复2-3次
  "完播率，完播率，还是完播率"
```

## 格式规范

### 视频规格

| 项目 | 规范 |
|------|------|
| 比例 | 竖屏 9:16（1080×1920） |
| 格式 | MP4 / MOV |
| 大小 | ≤4GB |
| 帧率 | 30fps / 60fps |
| 编码 | H.264 推荐 |

### 时长分级

| 时长 | 适用场景 | 难度 |
|------|----------|------|
| ≤15s | 热梗、快节奏、种草 | ★ |
| 15-60s | 教程、口播、故事 | ★★ |
| 1-3min | 深度教程、剧情、vlog | ★★★ |
| 3-10min | 专题、评测、长故事 | ★★★★ |

### 封面规范

```
尺寸: 1080×1440（3:4）或 1080×1920（9:16）
要素:
  - 大字标题（≤8字）
  - 高对比度
  - 人物出镜（有人脸封面点击率+30%）
  - 关键数字突出显示

避免:
  - 纯黑/纯白背景
  - 文字太小
  - 信息过载
```

### 字幕规范

```
位置: 画面下方 1/3
字体: 黑体/加粗，白色+黑色描边
字号: ≥24pt（确保手机端清晰）
样式: 关键词用不同颜色高亮
时间: 与口播严格同步
```

## 标题规则

### 视频标题（描述区）

| 规范 | 说明 |
|------|------|
| 字数 | 20-55 字（太短搜索权重低，太长显示不全） |
| 关键词 | 前 15 字必须包含核心关键词 |
| 格式 | #话题 + 正文描述 |
| 悬念 | 留白，不要在标题里剧透全部内容 |

### 标题公式

```
#话题标签 + Hook式标题
  "#自媒体运营 零基础也能做到的3个涨粉技巧，第3个很多人不知道"

#话题标签 + 痛点+解决
  "#短视频运营 总是没有播放量？这个方法我用了一周涨了5万粉"

#话题标签 + 反差悬念
  "#副业赚钱 他辞职做抖音，第一个月收入让所有人沉默了"
```

## 标签策略

### 话题标签（# 开头）

```
数量: 2-5 个
构成:
  1个行业大标签: #自媒体运营
  1-2个内容标签: #涨粉技巧 #短视频教程
  1个热门标签:   #抖音小助手（官方流量扶持标签）
  0-1个挑战标签: 当前热门挑战赛

位置: 视频描述最前面
```

### 官方流量标签

```
通用: #抖音小助手 #热门
领域: #知识分享 #干货分享 #创业 #科技
活动: 参与官方话题活动可获得额外曝光
```

## 发布时间

### 最佳时段

| 时段 | 流量级别 | 适合内容 |
|------|----------|----------|
| 7:00-8:00 | ★★★ | 早间资讯、励志 |
| 12:00-13:00 | ★★★★★ | 午休刷机高峰 |
| 17:00-19:00 | ★★★★ | 下班通勤 |
| 20:00-23:00 | ★★★★★ | 黄金时段 |

### 发布频率

```
新账号: 每天 1-2 条，保持活跃
成长期: 每天 1 条精品 > 每天 3 条凑数
稳定期: 稳定频率，培养用户期待
```

## 审核规避

详见 → `references/content-rules.md`

### 双重审核机制

```
第一轮: 机器审核（发布后即时）
  - AI 识别画面内容
  - 语音转文字后审核
  - 标题/描述关键词审核
  - 音乐版权审核

第二轮: 人工审核（流量触发）
  - 进入更大流量池时触发
  - 更细致的内容判断
  - 可能推翻机审结果
```

### 高危内容

| 类别 | 风险 | 处理 |
|------|------|------|
| 医疗建议 | 极高 | 必须标注"非专业建议" |
| 收益承诺 | 极高 | 禁用"月入XX""保证赚钱" |
| 版权音乐 | 高 | 使用抖音商用音乐库 |
| 竞品提及 | 中 | 避免直接 @竞品或说竞品名 |
| 站外引流 | 高 | 不能出现微信号/二维码 |
| 未成年人 | 极高 | 涉及未成年人内容需格外谨慎 |

### 限流触发条件

```
确认限流:
  - 视频发布后 2 小时播放量 < 100
  - 连续多条视频播放异常低

可能原因:
  - 含违禁词/敏感词
  - 搬运/重复发布
  - 账号权重被降（历史违规）
  - 内容质量太低（完播率极低）
  - 硬广/过度营销

恢复方法:
  - 删除违规视频
  - 暂停发布 3-7 天
  - 发布高质量原创内容
  - 参与官方活动
```

## 算法适配

详见 → `references/algorithm-guide.md`

### 核心指标权重

```
权重排序:
  1. 完播率    — 王者指标（前3秒+整体节奏）
  2. 互动率    — 点赞+评论+转发+收藏
  3. 转发率    — 高权重互动
  4. 评论率    — 高权重互动
  5. 关注转化率 — 从内容到关注的转化
  6. 点赞率    — 基础互动
```

### 五级流量池

```
第1级: 300-500 播放（初始推荐）
  目标: 完播率 > 30%，互动率 > 3%
  ↓ 达标升级
第2级: 3000-5000 播放
  目标: 完播率 > 25%，互动率 > 5%
  ↓ 达标升级
第3级: 1万-5万 播放
  目标: 数据持续优异
  ↓ 达标升级
第4级: 10万-50万 播放
  ↓ 达标升级
第5级: 100万+ 播放（热门）
```

### 完播率优化策略

| 位置 | 策略 |
|------|------|
| 0-3s | 强 Hook（痛点/成果/反常识），留住观众 |
| 3-10s | 承诺价值（"看完你就知道..."），建立期待 |
| 中段 | 信息密度高+节奏变化，防止走神 |
| 结尾前 | 投放悬念/彩蛋预告，鼓励看完 |
| 最后 | CTA + 引导完播（"最后一步最重要"） |
