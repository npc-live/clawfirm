# 抖音内容规则与审核

## 社区自律公约核心

### 禁止内容

1. **违法违规**：涉政敏感、暴力恐怖、色情低俗
2. **虚假信息**：假新闻、伪科学、虚假营销
3. **侵权行为**：版权侵犯、隐私泄露、肖像权侵犯
4. **不当营销**：虚假宣传、欺诈引流、传销推广
5. **未成年人保护**：涉及未成年人的不当内容
6. **危险行为**：鼓励/展示危险动作、极端行为

---

## 双重审核机制详解

### 第一轮：机器审核（即时）

```
审核内容:
  1. 视频画面: AI识别暴力/色情/违禁品等
  2. 音频内容: 语音转文字后关键词审核
  3. 文字内容: 标题+描述+字幕关键词
  4. 音乐: 版权库比对
  5. 画面特征: 其他平台水印/logo检测

时间: 发布后几秒到几分钟
结果:
  ✅ 通过 → 进入初始流量池
  ⚠️ 疑似违规 → 转人工审核（先不分发）
  ❌ 明确违规 → 直接删除/限流
```

### 第二轮：人工审核（触发式）

```
触发条件:
  - 机审标记为"疑似"
  - 视频进入更大流量池
  - 被用户举报
  - 定期巡检抽查

审核重点:
  - 内容上下文理解
  - 擦边/暗示性内容判断
  - 软广识别
  - 版权细致核查
```

## 禁用词分类

### 政治敏感词

```
红线: 涉及国家领导人、政治事件、敏感历史事件
处理: 零容忍，直接删除+警告/封号
注意: 即使是正面内容也要谨慎
```

### 违禁商品/服务

```
禁止推广:
  - 处方药、保健品夸大宣传
  - 烟草、电子烟
  - 赌博、博彩
  - 枪支、管制刀具
  - 野生动物制品
  - 虚拟货币/ICO
```

### 绝对化用语

```
禁用: 最、第一、唯一、极致、顶级、万能、绝对
     100%有效、包治百病、永久、无敌、全网最低

替代:
  "最好用" → "我用下来感觉很不错"
  "100%有效" → "亲测有效果"
  "全网最低" → "性价比很高"
```

### 收益承诺类

```
禁用: 月入{具体金额}、日赚{金额}、保证赚钱、
     躺赚、暴富、零成本、稳赚不赔、包赚

适用: 所有涉及副业/赚钱/创业的内容

替代:
  "月入10万" → "收入有了明显提升"
  "保证赚钱" → "我的个人经验"
  "零成本" → "投入很低"
```

### 站外引流词

```
禁用: 微信、wx、VX、威信、v信
     QQ、加我、私聊、二维码
     淘宝链接、拼多多链接

处理: 限流/删除视频，严重者封号

替代方案:
  - 抖音粉丝群
  - 抖音私信
  - 主页"一句话介绍"（有字数限制但可利用）
  - 抖音小店/橱窗（官方电商渠道）
```

## 版权规则

### 音乐版权

```
安全选择:
  - 抖音商用音乐库（内置）
  - 已获授权的音乐
  - 无版权音乐（CC0）

风险行为:
  - 使用未授权的流行音乐
  - 使用其他平台带水印的音频
  - 翻唱/改编未获授权

后果:
  - 视频被静音
  - 视频被删除
  - 账号收到版权警告
```

### 视觉素材版权

```
禁止:
  - 其他平台水印
  - 未授权的影视片段
  - 他人原创视频搬运
  - 未授权的品牌logo

允许:
  - 原创拍摄/制作
  - 已获授权的素材
  - 官方提供的特效/滤镜
```

## 创作者等级规则

### 信用分机制

```
满分: 100分
扣分项:
  - 违规内容: -5 到 -20 分/次
  - 虚假信息: -10 到 -20 分/次
  - 侵权: -10 分/次
  - 刷量: -20 分

低于 60 分:
  - 限制发布频率
  - 降低推荐权重
  - 严重者封号

恢复: 持续发布优质合规内容可逐步恢复
```

## 广告/商业合作规范

### 星图平台（官方合作）

```
品牌合作必须通过星图平台:
  1. 品牌在星图下单
  2. 达人接单并创作
  3. 内容标注"广告"
  4. 发布后数据报告

不走星图的风险:
  - 视频限流
  - 账号扣分
  - 品牌和达人双方处罚
```

### 直播带货规则

```
必须:
  - 开通抖音小店/橱窗
  - 商品需在白名单内
  - 不能虚假宣传
  - 不能诱导刷礼物
```
