# 抖音格式规范

## 视频参数

### 基础规格

| 项目 | 规范 |
|------|------|
| 比例 | 9:16 竖屏（推荐）/ 16:9 横屏 / 1:1 |
| 分辨率 | 1080×1920（竖屏）/ 1920×1080（横屏） |
| 格式 | MP4 / MOV |
| 大小 | ≤4GB |
| 帧率 | 30fps（推荐）/ 60fps |
| 编码 | H.264（推荐）/ H.265 |
| 码率 | ≥4Mbps（1080p） |
| 音频 | AAC, 44100Hz, ≥128kbps |

### 时长分级

| 时长 | 解锁条件 | 推荐场景 |
|------|----------|----------|
| ≤15s | 默认 | 热梗、快节奏、种草 |
| ≤60s | 默认 | 教程、口播、故事 |
| ≤3min | 1000粉丝 | 深度教程、剧情 |
| ≤10min | 申请长视频权限 | 专题、评测 |
| ≤30min | 特殊权限 | 直播回放、长专题 |

## 封面规范

### 封面尺寸

```
推荐: 1080×1440（3:4 比例）
  - 这是抖音封面的展示比例
  - 即使视频是 9:16，封面展示时会裁剪为 3:4

全屏: 1080×1920（9:16 比例）
  - 仅在个人主页以全屏预览时展示
```

### 封面设计

```
必须元素:
  - 大字标题（≤8字）
  - 字体加粗 + 描边/阴影
  - 人物出镜（有人脸的封面点击率更高）
  - 高对比度（手机小屏也能看清）

推荐风格:
  - 亮色背景 + 深色文字
  - 人物表情夸张/有感染力
  - 关键数字用大号+不同颜色

避免:
  - 纯黑/暗色背景（在feed中不突出）
  - 文字太小
  - 信息过多
  - 其他平台水印
```

## 字幕规范

### 字幕参数

| 项目 | 规范 |
|------|------|
| 位置 | 画面中下方（避免被底部UI遮挡） |
| 字体 | 黑体/微软雅黑，加粗 |
| 颜色 | 白色 + 黑色描边（2px） |
| 大小 | ≥24pt |
| 行数 | 同屏最多 2 行 |

### 字幕安全区域

```
竖屏 (1080×1920):
  - 顶部安全区: 距顶 ≥120px（状态栏）
  - 底部安全区: 距底 ≥300px（互动按钮+昵称）
  - 左右安全区: 距边 ≥40px

字幕推荐位置:
  - Y: 1200-1500px（画面中下部）
  - 居中对齐
```

### 字幕样式

```
普通文字: 白色 + 黑色描边
关键词高亮: 黄色/红色/绿色（与背景对比色）
强调: 加大字号 + 颜色变化
数字: 用不同颜色突出
```

## 音频规范

### 背景音乐

```
来源（安全）:
  - 抖音内置商用音乐库
  - 已购买版权的音乐
  - CC0 无版权音乐
  - 原创音乐

音量:
  - 背景音乐: 总音量的 20-30%
  - 口播/人声: 总音量的 70-80%
  - 确保人声清晰可听

注意:
  - 热门BGM可以蹭流量（平台会推荐同BGM内容）
  - 但版权音乐需确认可商用
```

### 人声录制

```
推荐:
  - 使用领夹麦/指向性麦克风
  - 采样率: 44100Hz / 48000Hz
  - 位深: 16bit
  - 格式: WAV（录制）→ AAC（导出）
  - 降噪处理

避免:
  - 手机内置麦克风直录（噪音大）
  - 音量忽大忽小
  - 回声/混响（除非刻意效果）
```

## 画面质量

### 拍摄要求

```
推荐:
  - 光线充足（自然光/补光灯）
  - 画面稳定（三脚架/云台）
  - 背景整洁
  - 色彩饱和度适中

避免:
  - 画面模糊/抖动
  - 逆光导致人脸黑
  - 背景杂乱
  - 过度滤镜（影响画面真实感）
```

### 剪辑规范

```
节奏:
  - 前3秒不能有静态画面（留住观众）
  - 每3-5秒一个镜头切换
  - 避免连续超过10秒的固定机位
  - 结尾不要黑屏/空白

转场:
  - 简洁为主（硬切 > 花式转场）
  - 避免过度使用特效转场
  - 节奏匹配音乐节拍
```

## 描述区规范

### 文字描述

| 项目 | 规范 |
|------|------|
| 字数 | ≤300字（推荐 20-55 字） |
| 格式 | #话题 + 正文 |
| 关键词 | 前15字包含核心关键词 |
| @功能 | 可 @相关账号 |
| 定位 | 可添加地理位置 |

### 描述模板

```
#话题标签 + 一句话Hook/价值点 + 互动引导

示例:
"#自媒体运营 3个涨粉技巧，第3个很多人不知道！你觉得哪个最实用？评论区告诉我"
```
