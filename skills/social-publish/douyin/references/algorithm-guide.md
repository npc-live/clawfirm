# 抖音推荐算法

## 核心机制：完播率至上

### 指标权重

```
权重排序:
  1. 完播率     ★★★★★  最核心指标
  2. 互动率     ★★★★   点赞+评论+转发+收藏
  3. 转发率     ★★★★   高价值互动
  4. 评论率     ★★★★   深度互动
  5. 点赞率     ★★★    基础互动
  6. 关注转化率  ★★★    内容→关注
  7. 收藏率     ★★     长期价值
  8. 下载率     ★★     传播力
```

### 完播率详解

```
计算: 完播率 = 完整观看次数 / 总播放次数

分级:
  > 40% — 优秀（大概率进入下一级流量池）
  30-40% — 良好
  20-30% — 一般
  < 20% — 较差（可能被限流）

注意:
  - 短视频（<15s）完播率要求更高
  - 长视频（>60s）完播率阈值更低
  - 重复播放也计入（循环播放是加分项）
```

## 五级流量池机制

### 详细说明

```
第1级: 初始推荐池（300-500播放）
  触发: 视频发布后自动进入
  分发: 粉丝+少量同标签用户
  评判时间: 1-2小时
  升级条件:
    - 完播率 > 30%
    - 互动率 > 3%（互动数/播放数）
    - 正向评论占比高

第2级: 扩大推荐池（3000-5000播放）
  触发: 第1级数据达标
  分发: 扩大到更多兴趣用户
  评判时间: 6-12小时
  升级条件:
    - 完播率 > 25%
    - 互动率 > 5%
    - 转发率明显

第3级: 中等推荐池（1万-5万播放）
  触发: 第2级数据持续优异
  分发: 跨标签推荐
  评判时间: 12-24小时
  升级条件:
    - 数据增速不减
    - 负面反馈率低

第4级: 大推荐池（10万-50万播放）
  触发: 内容质量+数据双优
  分发: 广泛推荐
  评判时间: 24-72小时

第5级: 热门/全网推荐（100万+播放）
  触发: 极优数据+话题热度
  分发: 全平台推荐
  特征: 可能上热搜、被官方推荐
```

### 流量池回退

```
注意: 流量池不是单向的
  - 进入高级流量池后如果数据下降，会被回退
  - 互动率突然降低 → 减少推荐
  - 负面反馈增加 → 可能限流

预防:
  - 确保内容质量稳定
  - 不要为了数据做标题党（高点击低完播）
  - 评论区积极回复维持互动
```

## DOU+ 投放逻辑

### DOU+ 与自然流量

```
DOU+ 的本质: 付费进入更大的流量池
  - 不改变内容质量评判标准
  - 只是加速分发过程
  - 内容差的视频投DOU+也不会变好

正确使用:
  1. 先观察自然流量数据（2-6小时）
  2. 如果完播率+互动率达标但曝光不够 → 值得投
  3. 如果完播率很低 → 不值得投（钱打水漂）

投放策略:
  - 小额测试（100元）→ 观察ROI → 决定是否加投
  - 选择"智能推荐" > "自定义人群"（除非很明确受众）
  - 黄金时段投放（20:00-23:00）效果最好
```

## 标签与话题权重

### 话题标签的算法作用

```
作用:
  1. 定义内容标签 → 匹配目标用户群
  2. 参与话题活动 → 额外曝光加成
  3. 搜索入口 → 长尾流量

最佳实践:
  - 选择与内容高度相关的话题
  - 1-2个精准话题 > 5个泛话题
  - 参与官方发起的话题活动（有流量扶持）
  - #抖音小助手 等通用标签效果已大幅下降
```

## 账号权重

### 影响账号权重的因素

```
正面:
  + 实名认证
  + 持续活跃（稳定发布）
  + 垂直领域（不频繁切换赛道）
  + 历史数据好（爆款视频数量）
  + 粉丝活跃度高
  + 直播互动

负面:
  - 违规记录
  - 长期不活跃
  - 频繁删除视频
  - 搬运/低质量内容
  - 刷量行为
  - 关联账号违规
```

### 新号冷启动

```
冷启动策略:
  1. 完善账号信息（头像、昵称、简介、认证）
  2. 前5条视频决定账号标签（务必垂直）
  3. 每条视频间隔 4-6 小时
  4. 前7天不要投DOU+（先建立自然标签）
  5. 积极互动（评论同领域视频）
  6. 前期追求完播率 > 播放量
```

## 时间权重

```
发布时间影响:
  - 发布后 1-2 小时是黄金数据窗口
  - 这个窗口内的数据决定是否升级流量池
  - 在用户活跃高峰发布可以获得更好的初始数据

最佳发布时间:
  - 午休: 12:00-13:00
  - 黄金: 20:00-23:00
  - 避免: 凌晨2:00-6:00（活跃用户少）
```
