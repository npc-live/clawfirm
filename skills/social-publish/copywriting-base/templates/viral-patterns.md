# 病毒传播模式

## 六大传播触发器

### 1. 社交货币（Social Currency）

让分享者看起来更好：

| 策略 | 说明 | 示例 |
|------|------|------|
| 内部人 | 分享独家/小众信息 | "很少人知道的Chrome插件" |
| 品味信号 | 体现审美/认知水平 | "这个设计理念太超前了" |
| 成就展示 | 让人展示自己的成果 | "晒出你的{主题}作品" |
| 先知先觉 | 提前知道趋势 | "2024年{领域}3大趋势预测" |

**应用：** 创作时问自己"用户转发这条内容后，会让 TA 在朋友眼中显得怎样？"

### 2. 触发物（Triggers）

与日常事物关联，反复被想起：

| 策略 | 说明 | 示例 |
|------|------|------|
| 时间锚点 | 绑定日常时间节点 | "每天早上的第一件事" |
| 场景关联 | 绑定常见场景 | "开会时老板说了这句话" |
| 热点借势 | 绑定当前热点 | "{热点事件}给我们的启示" |
| 习惯绑定 | 绑定日常习惯 | "刷手机时看到这个就赚了" |

**应用：** 内容主题与受众的高频场景/习惯挂钩。

### 3. 情绪驱动（Emotion）

高唤醒情绪驱动分享：

```
高唤醒（促进分享）         低唤醒（抑制分享）
├── 敬畏/惊叹 ★★★★★      ├── 满足 ★
├── 兴奋/激动 ★★★★★      ├── 悲伤 ★★
├── 愤怒/不公 ★★★★★      └── 平静 ★
├── 焦虑/紧迫 ★★★★
├── 幽默/快乐 ★★★★
└── 恐惧/担忧 ★★★
```

**应用：** 选择高唤醒情绪作为内容的情感基调。不要"温吞水"。

### 4. 公开性（Public）

让行为可见，触发模仿：

| 策略 | 说明 |
|------|------|
| 挑战赛 | "#30天{主题}挑战" |
| 晒单/晒图 | "晒出你的{主题}" |
| 模板分享 | 提供可填空的模板 |
| 打卡机制 | 每日/每周打卡 |

### 5. 实用价值（Practical Value）

有用就会被收藏和转发：

| 类型 | 分享动机 | 最佳形式 |
|------|----------|----------|
| 教程 | "以后用得上" | 步骤清单 |
| 工具 | "这个好用" | 对比测评 |
| 模板 | "拿来就能用" | 可下载资源 |
| 避坑 | "别人别踩坑" | 血泪经验 |
| 省钱 | "帮朋友省钱" | 折扣/免费替代 |

### 6. 故事性（Stories）

信息嵌入故事中随故事传播：

```
经典叙事弧线:
  困境/起点 → 尝试/挣扎 → 转折/发现 → 成果/蜕变

要素:
  - 具体细节（时间、地点、数字）
  - 情感起伏（低谷 → 高峰）
  - 可识别的主人公（最好是"我"）
  - 意外转折
  - 可复制的教训
```

## 病毒系数优化公式

```
病毒系数 K = 邀请数 × 转化率

K > 1 → 指数增长（自传播）
K = 1 → 线性增长
K < 1 → 衰减（需持续投入）
```

### 提高 K 的策略

| 阶段 | 策略 |
|------|------|
| 触达 | 标题+封面决定点击率 |
| 留存 | Hook + 节奏决定完读/完播率 |
| 分享 | 情绪+实用价值决定转发率 |
| 转化 | CTA 清晰度决定关注率 |

## 内容模板：病毒式结构

### 模板 A：震惊反转型

```
Hook: 反常识开场
正文:
  1. 展示"常识"
  2. 用数据/案例推翻
  3. 揭示真相
  4. 给出正确做法
CTA: "你之前也是这么想的吗？评论区聊聊"
```

### 模板 B：干货清单型

```
Hook: "整理了{N}个{主题}，先收藏再看"
正文:
  1. 要点1（最惊艳的放第一个）
  2. 要点2
  ...
  N. 要点N（压轴）
CTA: "还有什么补充？评论区一起完善"
```

### 模板 C：个人故事型

```
Hook: "去年这个时候我还在{低谷}"
正文:
  1. 困境描述（具体、可感）
  2. 转折点（发现了什么）
  3. 行动过程（做了什么）
  4. 成果展示（数据、对比）
  5. 复盘总结（可复制的方法）
CTA: "你也经历过类似的事吗？"
```

### 模板 D：对比测评型

```
Hook: "{A} vs {B}，到底选哪个？"
正文:
  1. 测评维度说明
  2. 逐维度对比（用表格/图示）
  3. 优缺点总结
  4. 推荐场景
CTA: "你更倾向哪个？投票/评论"
```
