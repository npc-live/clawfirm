# Hook 公式库

## 视频 Hook（前 3 秒定生死）

### 痛点直击型

```
"你是不是也{痛点}？"
"如果你正在{困境}，这条视频一定要看完"
"{痛点}？那是因为你还不知道{核心信息}"
"还在{错误做法}？难怪{负面结果}"
```

**示例：**
- "你是不是也发了100条都没什么播放量？"
- "还在每天花3小时剪视频？难怪你坚持不下去"
- "如果你的小红书笔记总是没人看，这条一定要看完"

### 成果展示型

```
"我用{方法}{时间}做到了{成果}"
"从{起点}到{终点}，我只做了这{数字}件事"
"{成果}，方法其实特别简单"
"我的{指标}是如何从{低}到{高}的"
```

**示例：**
- "我用这套模板7天涨了2万粉"
- "从0粉到10万，我只做了这3件事"
- "月入5万，方法其实特别简单"

### 反常识型

```
"{常识}？大错特错！"
"99%的人都在{错误做法}，真正有用的是{正确做法}"
"我要说的话可能得罪人，但{反常识观点}"
"{权威说法}？其实{真相}"
```

**示例：**
- "努力涨粉就能赚钱？大错特错！"
- "99%的人都在追热点，真正有用的是深度内容"

### 悬念制造型

```
"接下来的内容可能颠覆你的认知"
"这个方法我犹豫了很久要不要分享"
"看到最后有惊喜"
"{数字}个月前我发现了一个秘密"
```

### 利益承诺型

```
"看完这条你就知道{利益}"
"这条视频价值{金额}，免费分享给你"
"学会这招，{利益}"
"今天分享{数字}个{干货}，建议先收藏"
```

## 图文 Hook（前 2 行是生命线）

### 故事开场

```
"去年这个时候，我还在{低谷状态}..."
"3年前的今天，我做了一个决定..."
"说一个我从没公开说过的经历..."
"上个月发生了一件事，彻底改变了我的{方面}..."
```

### 清单预告

```
"整理了{数字}个{主题}，建议收藏 📌"
"全网最全{主题}合集，一篇搞定"
"把我{时间}的{领域}经验浓缩成{数字}条"
"{主题}避坑清单，每条都是血泪教训"
```

### 共情开场

```
"如果你也{处境}，这篇一定要看完"
"想对所有{身份}说一句话..."
"终于有人把{话题}说透了"
"这篇写给所有{处境}的人"
```

### 数据冲击

```
"{惊人数据}，而大多数人还不知道"
"刷到这条的概率只有{百分比}，说明你{暗示}"
"最新数据：{数据}，你准备好了吗？"
```

## Hook 组合技巧

### 双重 Hook（效果加倍）

```
痛点 + 利益: "还在为{痛点}发愁？这{数字}个方法直接解决"
反常识 + 悬念: "{常识}是错的，真相你可能接受不了"
成果 + 教程: "靠这个方法月入{金额}，今天把方法免费分享"
```

### Hook 自检清单

- [ ] 是否在 3 秒内传达核心信息？
- [ ] 是否触发了至少一种情绪（好奇/焦虑/兴奋/愤怒）？
- [ ] 是否与目标受众直接相关？
- [ ] 是否有继续看下去的理由？
- [ ] 是否避免了"大家好我是xxx"这种无效开场？
