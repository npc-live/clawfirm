# CTA 话术库

## 按互动目标分类

### 关注引导

```
温和型:
  "关注我，持续分享{主题}干货"
  "点个关注不迷路，下次更新不错过"

利益型:
  "关注我，下期教你{下期预告}"
  "关注 @{账号} 获取{资源}"

紧迫型:
  "我每周只更新{N}条，关注防走丢"
  "这个系列还有{N}期，关注追更"
```

### 点赞引导

```
直接型:
  "觉得有用就点个赞吧 👍"
  "花了{时间}整理，点赞是最好的鼓励"

互惠型:
  "点赞过{N}我就出下期教程"
  "你的点赞是我持续更新的动力"

心理暗示:
  "看到这里的都是{正面标签}的人，顺手点个赞"
```

### 收藏引导

```
实用型:
  "先收藏，用到的时候不迷路 📌"
  "建议收藏，总有一天你会用到"
  "码住！{时间节点}肯定能用上"

清单型:
  "这份{主题}清单值得反复看"
  "收藏这篇 = 收藏一份{主题}指南"
```

### 评论引导

```
问题型:
  "你觉得{话题}怎么样？评论区聊聊"
  "你遇到过{问题}吗？"
  "你是{选项A}还是{选项B}？"

投票型:
  "同意的扣1，不同意的扣2"
  "你属于哪种类型？评论区告诉我"

补充型:
  "还有什么我没提到的？评论区补充"
  "你的经验是什么？一起交流"

故事型:
  "说说你的{主题}故事"
  "你最{形容词}的一次{经历}是什么？"
```

### 转发引导

```
利他型:
  "转给你身边正在{处境}的朋友"
  "转发给需要的人"

价值型:
  "这篇干货值得分享给你的{关系人}"
  "转发 = 帮你的{关系人}避坑"
```

### 私信/加群引导

```
资源型:
  "想要{资源}的扣'1'，私信发你"
  "完整版{资源}私信'关键词'领取"

社群型:
  "加入我们的{主题}交流群，评论区留言"
  "想和{N}+{身份}一起成长？评论'进群'"
```

## CTA 组合公式

### 单目标 CTA

只要一个动作，转化率最高：

```
"{简短感谢/总结} + {一个明确动作}"
→ "觉得有用就关注我，每天分享运营干货"
```

### 双目标 CTA

```
"{动作1} + {动作2}，{利益}"
→ "点赞 + 关注，下期教你写爆款标题"
→ "收藏 + 转发，帮助更多人避坑"
```

### 三连 CTA

```
"一键三连（点赞+收藏+关注），你的支持是我更新的动力 🙏"
"如果这条对你有帮助，点赞收藏关注三连，下期更精彩"
```

### 互动 + 关注 CTA

```
"{互动问题}？评论区聊聊，顺手点个关注，下期接着讲"
"你属于哪种？评论告诉我 + 关注看后续分析"
```

## 平台适配要点

| 平台 | CTA 重点 | 禁忌 |
|------|----------|------|
| 小红书 | 收藏 > 点赞 > 关注 | 不能明确引导站外（微信号等） |
| Twitter | RT > Reply > Like | 避免过于推销的语气 |
| 抖音 | 关注 > 评论 > 点赞 | 不能引导关注竞品账号 |
| B站 | 三连 > 弹幕 > 关注 | 不能硬广，要"恰饭"预告 |

## CTA 优化清单

- [ ] CTA 是否在内容结尾？（最佳位置）
- [ ] CTA 是否只有一个核心动作？（别贪多）
- [ ] CTA 是否给出了明确的行动理由？
- [ ] CTA 的措辞是否符合平台调性？
- [ ] CTA 是否避开了平台禁忌？
- [ ] CTA 是否创造了紧迫感或稀缺性？
