## 标题公式

### 1. 数字型
用具体数字建立可信度和可预期性。

| 公式 | 示例 |
|------|------|
| {数字} + {结果} + {时间} | "7天涨粉1万的3个核心动作" |
| {数字}个{方法}让你{结果} | "5个方法让你写作效率翻倍" |
| 从{起点}到{终点}只用了{时间} | "从0到10万粉只用了60天" |
| {数字}%的人不知道的{秘密} | "90%的人不知道的涨粉秘密" |

### 2. 疑问型
用问题激发好奇心，触发点击。

| 公式 | 示例 |
|------|------|
| 为什么{现象}？ | "为什么你的内容总是没人看？" |
| {行为}真的有用吗？ | "每天发3条真的有用吗？" |
| 你知道{事实}吗？ | "你知道算法在前3秒就决定了你的命运吗？" |
| 如何{目标}？（完整指南） | "如何从零开始做自媒体？（2024完整指南）" |

### 3. 反差型
用出乎意料的对比制造认知冲突。

| 公式 | 示例 |
|------|------|
| {低起点}却{高成就} | "高中学历却年入百万" |
| 别再{常见做法}了，{正确做法} | "别再追热点了，深度内容才是王道" |
| {看似正确}其实{真相} | "日更看似勤奋其实是最大的懒惰" |
| 我{负面经历}后{正面转折} | "被裁员后我月入翻了3倍" |

### 4. 恐惧型
用损失规避心理驱动行动。

| 公式 | 示例 |
|------|------|
| {不做X}你会{后果} | "不懂这个规则你会被永久限流" |
| {身份}必须避开的{数字}个坑 | "新手必须避开的5个致命坑" |
| 千万不要{行为} | "千万不要在这个时间段发视频" |
| {后果}的{数字}个信号 | "账号即将被限流的3个信号" |

### 5. 利益型
直接点明用户能获得什么。

| 公式 | 示例 |
|------|------|
| {动作}即可获得{利益} | "收藏这篇就够了，小红书运营全攻略" |
| 免费{资源}（{限制条件}） | "免费模板分享（限时3天）" |
| {结果}模板直接套用 | "爆款标题模板直接套用" |
| 一文搞定{问题} | "一文搞定所有平台的发布规范" |

## Hook 开场公式

前 3 秒（视频）/ 前 2 行（图文）决定内容的生死。

### 视频 Hook（前 3 秒）

| 类型 | 公式 | 示例 |
|------|------|------|
| 痛点直击 | "你是不是也{痛点}？" | "你是不是也发了100条内容都没人看？" |
| 成果展示 | "我用{方法}{时间}做到了{成果}" | "我用这个方法30天涨了5万粉" |
| 反常识 | "{常识}？错！{真相}" | "努力发内容就能涨粉？错！" |
| 权威背书 | "作为{身份}，我告诉你{核心观点}" | "作为运营了3年的博主，我告诉你..." |
| 悬念制造 | "接下来我要说的，可能会让你{反应}" | "接下来的内容可能颠覆你的认知" |
| 利益承诺 | "看完这条，你就能{利益}" | "看完这条你就知道怎么写爆款标题了" |

### 图文 Hook（前 2 行）

| 类型 | 公式 |
|------|------|
| 故事开场 | "去年这个时候，我还在{低谷状态}..." |
| 清单预告 | "整理了{数字}个{主题}，建议收藏" |
| 共情开场 | "如果你也{处境}，这篇一定要看完" |
| 数据冲击 | "{惊人数据}，而大多数人还不知道" |

## 正文结构框架

### AIDA（注意-兴趣-欲望-行动）

适用：产品推广、种草内容

```
A - Attention: 抓注意力的开场（Hook）
I - Interest:  引发兴趣的信息/故事
D - Desire:    激发欲望的痛点/利益
A - Action:    明确的行动号召（CTA）
```

### PAS（问题-放大-解决）

适用：痛点驱动内容、教程

```
P - Problem:  提出问题/痛点
A - Agitate:  放大痛苦，描述不解决的后果
S - Solution: 给出解决方案
```

### BAB（之前-之后-桥梁）

适用：对比型内容、案例分享

```
B - Before: 使用前的痛苦状态
A - After:  使用后的理想状态
B - Bridge: 连接两者的方法/产品
```

### 列表体（适用社交平台）

```
标题（数字型最佳）
├── 要点 1 + 简要说明
├── 要点 2 + 简要说明
├── ...
└── 总结 + CTA
```

详见 → `templates/cta-templates.md`

### 按目标分类

| 目标 | 话术示例 |
|------|----------|
| 关注 | "关注我，每天分享{主题}干货" |
| 点赞 | "觉得有用就点个赞吧" |

### CTA 组合公式

```
单一 CTA: "关注我获取更多干货"
双重 CTA: "点赞 + 关注，下期更新{预告}"
三连 CTA: "觉得有用就点赞收藏关注三连吧"
互动 CTA: "你遇到过{问题}吗？评论区分享你的经验"
紧迫 CTA: "名额有限，先到先得"
```

## 病毒传播因子

内容能自发传播需要具备以下至少 2 个因子：

### 1. 情绪触发

| 情绪 | 传播力 | 适用场景 |
|------|--------|----------|
| 敬畏/惊叹 | ★★★★★ | 技术突破、不可思议的成果 |
| 愤怒/不公 | ★★★★★ | 行业黑幕、不合理现象 |
| 焦虑/恐惧 | ★★★★ | 风险预警、避坑指南 |
| 快乐/幽默 | ★★★★ | 段子、反转、意外结局 |
| 悲伤/共情 | ★★★ | 个人故事、情感连接 |

### 2. 社交货币

让转发者显得"有品位/有见识/有价值"：
- 独家信息（"我知道你不知道的"）
- 身份认同（"这就是我"）
- 实用工具（"转发备用"）

### 3. 实用价值

可直接操作的内容：
- 教程/指南/清单
- 模板/工具推荐
- 避坑/省钱/提效

### 4. 故事性

有叙事弧线的内容：
- 起点（困境）→ 转折（发现方法）→ 结果（成功）
- 越具体、越真实、越有细节，传播力越强

## 中英双语写作规范

### 核心原则：适配，不翻译

同一内容的中文版和英文版应该是"同一个意思的两种原生表达"，而非互相翻译。

### 中文内容规范

- 口语化 > 书面化（社交平台）
- 短句 > 长句（每句 ≤ 20 字为佳）
- 善用"你""我""咱"建立亲近感
- 数字用阿拉伯数字，不要"三"要"3"
- emoji 作为分隔符和情绪标记

### 英文内容规范

- Active voice > Passive voice
- Conversational tone for social
- Power words: free, new, proven, secret, ultimate, hack
- Sentence length: ≤ 15 words average
- Use contractions (don't, can't, it's)

### 中英差异对照

| 方面 | 中文 | English |
|------|------|---------|
| 开场 | "姐妹们听我说" | "Here's the thing..." |
| 数字 | "3个月涨粉10万" | "100K followers in 90 days" |
| CTA | "点赞收藏关注三连" | "Like + Follow for more" |
| 标签 | #自媒体运营 #涨粉 | #ContentCreator #GrowthHacks |
| 语气 | 分享、种草、安利 | Tips, hacks, game-changer |
