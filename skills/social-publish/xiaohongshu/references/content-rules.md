# 小红书内容规则与审核

## 社区规范核心

### 禁止行为

1. **虚假营销**：虚构使用体验、刷量刷赞
2. **站外引流**：直接展示微信号、QQ、手机号、二维码
3. **抄袭搬运**：搬运他人原创内容
4. **违规广告**：未标注的软广、伪装测评
5. **不当言论**：歧视、侮辱、造谣

### 禁用词分类

#### 绝对化用语（广告法红线）

```
禁用: 最好、最佳、最优、第一、唯一、首选、全网最低、
     绝对、100%、万能、顶级、极致、永久、无敌

替代方案:
  "最好" → "我觉得很不错" / "个人强推"
  "第一" → "排名前列" / "非常靠前"
  "100%" → "亲测有效" / "目前来看效果很好"
  "永久" → "长效" / "持久"
```

#### 医疗健康类禁用词

```
禁用: 治疗、治愈、药效、根治、处方、临床、特效药、
     包治百病、立竿见影、药到病除

适用: 任何涉及健康/美容/保健的内容
替代: "个人体验""仅供参考""建议咨询专业医生"
```

#### 金融理财类禁用词

```
禁用: 保本、保收益、稳赚不赔、日入{金额}、躺赚、
     零风险、高收益、翻倍、暴富

适用: 任何涉及赚钱/理财/投资的内容
替代: "我的个人经验""不构成投资建议""结果因人而异"
```

#### 引流相关禁用词

```
禁用: 微信号、VX、wx、威信、v信、加我、私聊转微
     QQ号、手机号、淘口令、拼多多链接

替代方案:
  - "看主页" / "主页有联系方式"
  - "私信回复'关键词'"
  - "小红书群聊"（平台内功能）
```

## 限流触发条件

### 确定限流信号

| 信号 | 说明 |
|------|------|
| 笔记曝光骤降 | 发布 2h 后阅读量 < 50 |
| 搜索不到笔记 | 用标题搜索找不到 |
| 别人看不到 | 其他用户访问主页看不到该笔记 |

### 限流原因

```
内容层面:
  - 含禁用词/敏感词
  - 内容涉嫌虚假宣传
  - 搬运/高相似度内容
  - 图片含其他平台水印
  - 内容违反社区规范

操作层面:
  - 发布后频繁编辑
  - 发布后删除再重发
  - 短时间内发布大量笔记
  - 使用第三方工具批量操作
  - 频繁修改昵称/头像/简介
```

### 限流恢复

```
轻度限流（单篇）:
  1. 检查并删除违规内容
  2. 修改后重新发布
  3. 下次避免同类问题

重度限流（账号级）:
  1. 停更 3-7 天
  2. 期间正常浏览+互动（证明不是机器人）
  3. 恢复发布后先发安全内容
  4. 逐步恢复正常发布频率
```

## 敏感领域特别注意

### 医疗/美容

```
✅ 可以: 分享个人体验（标注"个人感受，仅供参考"）
❌ 不可: 宣称疗效、推荐处方药、替代专业医疗建议
⚠️ 必须: 在笔记中标注免责声明
```

### 金融/理财

```
✅ 可以: 分享投资经验（标注"不构成投资建议"）
❌ 不可: 承诺收益、推荐非法金融产品
⚠️ 必须: 风险提示
```

### 教育/培训

```
✅ 可以: 分享学习方法和经验
❌ 不可: 虚假学历/资质、保过承诺
⚠️ 注意: 不要出现"保过""包就业"等绝对承诺
```

## 品牌合作规范

### 蒲公英平台（官方合作）

```
品牌合作必须通过蒲公英平台报备:
  1. 品牌在蒲公英下单
  2. 博主接单并创作
  3. 内容标注"品牌合作"
  4. 发布后报告数据

不报备的风险:
  - 笔记被限流
  - 账号被处罚
  - 品牌和博主双方受罚
```
