# 小红书格式规范

## 图片规范

### 尺寸

| 比例 | 尺寸 | 适用 |
|------|------|------|
| 3:4 | 1080×1440 | 推荐，信息流展示最大 |
| 1:1 | 1080×1080 | 通用，适合产品图 |
| 4:3 | 1440×1080 | 横图，适合对比图 |
| 16:9 | 1920×1080 | 横屏，展示面积较小 |

### 图片格式

| 项目 | 规范 |
|------|------|
| 格式 | JPG / PNG |
| 单张大小 | ≤20MB |
| 图片数量 | 1-18 张（推荐 6-9 张） |
| 分辨率 | ≥1080px 宽 |

### 封面图设计

```
第一张图 = 封面（决定点击率）

必须元素:
  - 大字标题（≤10字）
  - 字体加粗+描边/阴影
  - 高对比度配色
  - 核心信息一目了然

推荐配色:
  - 白底+黑字+彩色点缀（干净专业）
  - 莫兰迪色系（高级感）
  - 糖果色/马卡龙色（年轻可爱）

禁忌:
  - 其他平台水印
  - 二维码
  - 模糊/低画质
  - 文字太小看不清
```

### 图文排版建议

```
图片顺序:
  第1张: 封面（吸引点击）
  第2-N张: 核心内容（教程步骤/产品展示/对比图）
  最后1张: CTA（关注/收藏引导）

每张图:
  - 一个核心信息点
  - 文字不超过图片面积的 30%
  - 保持风格统一
```

## 视频规范

### 视频参数

| 项目 | 规范 |
|------|------|
| 比例 | 9:16（竖屏推荐）/ 16:9 / 1:1 |
| 分辨率 | 1080×1920（竖屏）/ 1920×1080（横屏） |
| 时长 | ≤15 分钟（推荐 1-3 分钟） |
| 格式 | MP4 / MOV |
| 大小 | ≤5GB |
| 帧率 | 30fps / 60fps |
| 编码 | H.264 推荐 |
| 码率 | ≥4Mbps |

### 视频封面

```
尺寸: 与视频比例一致
  竖屏: 1080×1440 或 1080×1920
  横屏: 1920×1080

可选:
  - 自动截取视频帧
  - 自定义上传封面图（推荐）
```

### 字幕要求

```
字幕: 强烈建议添加（很多用户静音浏览）
位置: 画面下方 1/3
字体: 无衬线+加粗
颜色: 白色+黑色描边
大小: ≥20pt
```

## 文字规范

### 正文

| 项目 | 规范 |
|------|------|
| 字数 | 300-1000 字（500-700 最佳） |
| 段落 | 每段 2-4 行，段间空行 |
| emoji | 每 2-3 句一个 emoji 作分隔 |
| 格式 | 支持加粗（用 ** 包裹） |

### SEO 关键词布局

```
标题: 核心关键词（权重最高）
正文第一段: 核心关键词（自然嵌入）
正文中: 长尾关键词 2-3 次
标签: 关键词+相关词
```

---

## 标签格式

```
格式: # + 标签文字（无空格）
数量: 3-8 个（推荐 5-6 个）
位置: 正文末尾
长度: 每个标签 2-8 字
```

### 标签示例

```
内容类:
  #小红书运营 #自媒体干货 #涨粉攻略

领域类:
  #程序员日常 #AI工具 #效率提升

热门类:
  #每日分享 #今日份分享 #干货分享
```
