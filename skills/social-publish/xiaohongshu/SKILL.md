# 小红书内容创作

## 文案风格

### 核心调性：姐妹种草、真诚分享

- **口语化**：像和闺蜜聊天，不要书面语
- **emoji 友好**：每 2-3 句用 emoji 分隔，但不堆砌
- **分享感**：用"我""亲测""真的推荐"等第一人称
- **种草感**：具体感受 > 抽象描述（"涂上嘴巴嘟嘟的" > "效果很好"）
- **真实感**：适当暴露小缺点，增加可信度

### 语气词库

```
常用: 真的绝了 / 姐妹们 / 救命 / 谁懂 / 绝绝子 / yyds / 宝子们
赞叹: 也太好了吧 / 爱了爱了 / 直接封神 / 这也太绝了
推荐: 闭眼入 / 人手一个 / 不允许还有人不知道 / 必须安利
转折: 但是！/ 划重点 / 注意⚠️ / 避雷！
```

### 文案结构模板

```
【标题】emoji + 关键词 + 数字/利益点
【正文】
  第1段: Hook（痛点/成果/反常识）
  第2段: 核心内容（列点/步骤/对比）
  第3段: 个人体验/使用感受
  第4段: CTA（收藏 > 关注 > 评论）
【标签】2核心 + 3长尾 + 1热门
```

## 格式规范

### 图文笔记（主力形式）

| 项目 | 规范 |
|------|------|
| 图片数量 | 6-9 张最佳（≥3 张，≤18 张） |
| 图片尺寸 | 3:4（1080×1440）推荐，1:1 也可 |
| 封面图 | 第一张即封面，决定点击率 |
| 文字字数 | 500-700 字最佳（≥300 字获得搜索权重） |
| 图片格式 | JPG/PNG，≤20MB/张 |

### 封面设计要点

```
必须元素:
  - 大字标题（≤10 字）
  - 高对比度（深色文字 + 浅色背景 或反之）
  - 关键信息可视化（数字、对比图、清单预览）

避免:
  - 纯文字无图（点击率低）
  - 文字过小（手机端看不清）
  - 信息过载（一张图一个核心信息）
```

### 视频笔记

| 项目 | 规范 |
|------|------|
| 比例 | 竖屏 9:16（1080×1920）推荐 |
| 时长 | 1-3 分钟最佳 |
| 格式 | MP4, ≤5GB |
| 字幕 | 必须有字幕，平台自动识别 + 手动校正 |
| 封面 | 可自定义，建议和图文封面同样设计 |

## 标题规则

### 基本规范

- 字数：≤20 字（含 emoji）
- 关键词前置（搜索权重集中在前 10 字）
- 至少包含 1 个数字
- 至少包含 1 个 emoji
- 禁用绝对化用词（"最好""第一""全网唯一"等）

### 标题公式（小红书特供）

| 公式 | 示例 |
|------|------|
| emoji + 数字 + 结果 | "🔥 3步搞定小红书爆款封面" |
| 身份 + 推荐 + 数字 | "设计师私藏的5个免费素材网站" |
| 反差 + 结果 | "零基础7天学会视频剪辑" |
| 痛点 + 解决 | "总是写不出好标题？这个公式直接套用" |
| 种草 + 限定 | "这个{品类}闭眼入！用了3年还在回购" |

### 标题禁区

```
❌ 绝对化: 最好、第一、全网唯一、100%
❌ 诱导互动: 必须点赞、不转不是中国人
❌ 虚假宣传: 月入百万（无法验证的收益承诺）
❌ 敏感词汇: 见 references/content-rules.md
```

## 标签策略

### 标签配比（5-8 个标签最佳）

```
2 个核心标签: 与内容主题直接相关（搜索流量）
3 个长尾标签: 细分领域/场景标签（精准匹配）
1 个热门标签: 当前平台热点话题（蹭流量）
1-2 个通用标签: #干货分享 #每日分享 等（泛流量）
```

### 标签发现方法

1. 搜索框联想词
2. 笔记详情页底部推荐标签
3. 热门话题广场
4. 竞品笔记标签分析

### 标签格式

```
正确: #小红书运营 #自媒体干货 #涨粉技巧
错误: #小红书运营技巧涨粉方法（太长，无法精准匹配）
```

## 发布时间

### 最佳时段

| 时段 | 流量级别 | 适合内容 |
|------|----------|----------|
| 7:00-9:00 | ★★★ | 早间灵感、日常分享 |
| 12:00-13:30 | ★★★★★ | 午休时段，全品类 |
| 18:00-20:00 | ★★★★ | 下班后，种草/购物 |
| 21:00-23:00 | ★★★★★ | 黄金时段，全品类 |

### 发布日历

- 周一至周四：稳定输出日
- 周五至周日：流量高峰，重点内容优先发
- 节假日前 1-2 天：节日相关内容提前布局

## 审核规避

详见 → `references/content-rules.md`

### 关键规则

1. **禁用词**：医疗断言、绝对化用语、政治敏感词、竞品诋毁词
2. **站外引流**：不能直接出现微信号、QQ 号、手机号
   - 替代方案：引导"看主页""私信回复关键词"
3. **敏感领域**：医疗、金融、法律内容需格外谨慎
4. **图片审核**：不能有其他平台水印、不能有二维码
5. **视频审核**：不能有其他平台 logo、不能有明显广告

### 限流信号

```
触发限流的行为:
  - 笔记含站外链接/联系方式
  - 大量使用禁用词
  - 短时间内多次编辑已发布笔记
  - 发布后立即删除重发
  - 被多人举报
  - 搬运他人内容
```

## 算法适配

详见 → `references/algorithm-guide.md`

### CES 评分模型

```
CES = 点赞(1) + 收藏(1) + 评论(4) + 转发(4) + 关注(8)

权重优先级: 关注 > 转发 ≈ 评论 > 点赞 ≈ 收藏
```

### 流量池机制

```
初始池: 200-500 曝光
  ↓ CES 达标
二级池: 2000-5000 曝光
  ↓ CES 达标
三级池: 1万-5万 曝光
  ↓ CES 达标
热门池: 10万+ 曝光
```

### 算法优化策略

| 指标 | 优化方法 |
|------|----------|
| 点击率 | 封面+标题优化 |
| 完读率 | 内容节奏+视觉排版 |
| 互动率 | CTA+争议话题+提问 |
| 收藏率 | 干货价值+清单格式 |
| 关注率 | 人设一致性+系列内容 |

```

📝 正文:
[500-700字正文，分段+emoji分隔]

🏷️ 标签:
#核心1 #核心2 #长尾1 #长尾2 #长尾3 #热门话题

⏰ 建议发布时间: [具体时段]

⚠️ 合规检查:
  - 禁用词: ✅ 无
  - 站外引流: ✅ 无
  - 绝对化用语: ✅ 无
```
