# B站推荐算法

## 核心指标

### 指标权重

```
B站特色权重（与其他平台不同）:

  1. 弹幕数    ★★★★★  B站独有的核心指标
  2. 投币数    ★★★★★  用户需消耗硬币，代表高认可
  3. 收藏数    ★★★★   内容长期价值
  4. 分享数    ★★★★   外部传播力
  5. 点赞数    ★★★    基础认可
  6. 播放量    ★★★    基础指标
  7. 完播率    ★★★    内容质量
  8. 评论数    ★★     社区互动

一键三连 = 点赞 + 投币 + 收藏（一次操作三个正面信号）
```

### 关键洞察

```
1. 弹幕 > 评论
   B站最独特的互动形式，权重极高
   引导观众发弹幕比引导评论更有效

2. 投币权重极高
   因为需要消耗用户资源（硬币），代表真实认可
   每人每天只能投有限的币

3. 收藏 = 长期价值信号
   B站的"收藏"等同于"以后还想看"
   教程类/工具类内容收藏率天然高

4. 完播率计算方式不同
   B站中视频较长，完播率阈值比短视频平台低
   看完 50% 以上就是不错的完播率
```

## 推荐机制

### 首页推荐

```
候选来源:
  1. 关注UP主的新视频
  2. 基于用户画像的兴趣推荐
  3. 热门/趋势内容
  4. 分区热门
  5. 编辑推荐

排序因子:
  - 内容质量分（互动数据综合）
  - 用户兴趣匹配度
  - 内容时效性
  - UP主权重
```

### 分区热门

```
每个分区有独立的热门排行:
  - 基于24小时内的综合数据
  - 数据增速 > 绝对数值（给新UP主机会）
  - 与分区标签匹配度

策略:
  - 选对分区是基础（内容与分区匹配）
  - 冷门分区竞争小，更容易上热门
  - 新分区流量扶持更多
```

### 搜索推荐

```
影响搜索排名:
  1. 标题关键词匹配
  2. 简介关键词匹配
  3. 标签匹配
  4. 视频互动数据
  5. UP主权重
  6. 发布时间（近期优先）

B站搜索特点:
  - 用户经常通过搜索找内容
  - 长标题有优势（可包含更多关键词）
  - 技术/教程类内容搜索流量占比高
```

## 每周必看/编辑推荐

```
"每周必看"是B站编辑团队手动挑选的优质内容:
  - 非算法推荐，是人工编辑
  - 被选中 = 巨大流量
  - 门槛很高（内容质量+制作水准）

入选条件（非官方，经验总结）:
  - 制作精良
  - 有深度/创意
  - 内容对社区有价值
  - UP主有一定基础（但新UP主也有机会）
```

## 账号权重

### UP主等级系统

```
等级影响推荐权重:
  Lv0-Lv3: 基础权重
  Lv4-Lv5: 中等权重
  Lv6:     高权重

升级途径:
  - 发布视频
  - 获得互动（弹幕/评论/投币等）
  - 参与社区活动
  - 日常活跃（观看/互动）
```

### 垂直度

```
垂直度对推荐的影响:
  - 持续在同一分区发布 → 算法更精准推荐
  - 频繁切换分区 → 用户画像混乱 → 推荐不精准
  - 建议: 一个账号专注1-2个领域

标签建立:
  - 前10-20个视频非常关键
  - 决定了算法给你的账号打什么标签
  - 标签一旦建立，短期内很难改变
```

## 互动优化策略

### 引导弹幕（最高权重）

```
技巧:
  - "弹幕打出'懂了'让我看看有多少人学会了"
  - "你觉得A还是B？弹幕告诉我"
  - "弹幕刷'这也太绝了'让我看看"
  - 在画面上设置弹幕触发点（出现某个画面时引导弹幕）

时机:
  - 每2-3分钟一个弹幕互动点
  - 关键节点/转折处引导
  - 开头就设置第一个弹幕互动
```

### 引导投币

```
技巧:
  - "觉得有用就投个币吧，是我继续更新的动力"
  - "说好的一键三连呢？"
  - 内容结尾自然引导

关键:
  - 内容必须真的有价值（B站用户不会给垃圾内容投币）
  - 投币= 用户愿意消耗自己的资源来奖励你
  - 不要过度请求
```

### 引导收藏

```
技巧:
  - 教程类: "建议收藏，用到的时候随时回来看"
  - 清单类: "收藏这个视频 = 收藏一份完整指南"
  - 系列类: "收藏这个合集，后续更新不错过"
```

## 内容生命周期

```
B站 vs 其他平台的区别:
  - B站内容的长尾效应非常强
  - 好的技术/教程视频可以持续获得流量数月甚至数年
  - 搜索流量是重要来源

生命周期:
  阶段1 - 爆发期（0-48h）: 主要靠推荐流量
  阶段2 - 增长期（2-7天）: 推荐+搜索
  阶段3 - 长尾期（7天+）: 主要靠搜索+推荐长尾

  特殊: 技术/教程类内容
    - 可以持续获得搜索流量
    - 发布半年后仍可能有大量播放
    - SEO优化（标题+标签）非常重要
```

## 创作者激励算法

```
激励计算:
  - 基于播放量 + 互动质量
  - 不是简单的"千次播放X元"
  - 高互动率的视频单次播放收益更高
  - 收益在稿件发布7天后结算

提高收益:
  - 提高互动率（弹幕、投币、收藏）
  - 增加视频时长（≥3分钟有更多广告位）
  - 保持更新频率
  - 参与激励计划（需满足粉丝/播放门槛）
```
