# B站格式规范

## 视频参数

### 中视频（推荐）

| 项目 | 规范 |
|------|------|
| 比例 | 16:9 横屏（推荐）/ 4:3 / 1:1 |
| 分辨率 | 1920×1080（推荐）/ 2560×1440 / 3840×2160 |
| 时长 | 1-60 分钟（推荐 3-10 分钟） |
| 格式 | MP4 / FLV / AVI / MOV / WMV / MKV |
| 大小 | ≤8GB |
| 帧率 | 30fps / 60fps / 120fps |
| 编码 | H.264（推荐）/ H.265 |
| 码率 | ≥6Mbps（1080p）/ ≥12Mbps（4K） |
| 音频 | AAC / MP3, ≥128kbps |

### 短视频（竖屏模式）

| 项目 | 规范 |
|------|------|
| 比例 | 9:16 竖屏 |
| 分辨率 | 1080×1920 |
| 时长 | ≤3 分钟 |
| 入口 | B站短视频 Tab / 动态 |
| 适用 | 碎片化内容、快节奏 |

## 封面规范

### 封面尺寸

```
推荐: 1920×1080（16:9）
最小: 960×600
文件: JPG / PNG
大小: ≤5MB
```

### 封面设计

```
B站封面特点（与其他平台不同）:
  - 可以承载更多文字信息
  - B站用户习惯"读"封面
  - 封面信息密度可以比抖音/小红书更高

必须元素:
  - 标题文字（≤12字，加描边/阴影）
  - 核心画面/人物

推荐元素:
  - 【】分类标签
  - 系列编号
  - 关键数字/数据
  - 品牌/系列logo

排版:
  - 标题在上方或中央
  - 文字+描边确保清晰
  - 留出底部空间（播放量/时长会覆盖底部）

避免:
  - 纯文字无画面
  - 过于花哨的设计
  - 文字被底部信息遮挡
  - 低分辨率/模糊
```

## 字幕规范

### 字幕格式

```
B站支持的字幕:
  1. 内嵌字幕（硬字幕，在视频中）
  2. 外挂字幕（SRT/ASS 格式上传）
  3. AI自动字幕（B站自动生成，可校正）

推荐: 内嵌字幕 + AI字幕双保险
```

### 内嵌字幕参数

| 项目 | 规范 |
|------|------|
| 位置 | 画面下方（距底部 10-15%） |
| 字体 | 微软雅黑/黑体，加粗 |
| 颜色 | 白色 + 黑色描边（2px） |
| 大小 | 38-48pt（1080p下） |
| 行数 | 同屏最多 2 行 |
| 字间距 | 适当加宽（提高可读性） |

### 外挂字幕（SRT格式）

```
格式: SRT / ASS / VTT
编码: UTF-8
时间码: 精确到毫秒
内容: 每段 ≤2 行，每行 ≤20 字

示例（SRT）:
  1
  00:00:01,000 --> 00:00:03,500
  大家好，今天来聊聊AI工具

  2
  00:00:04,000 --> 00:00:07,000
  我测试了10个工具，选出最好用的3个
```

## 标题规范

### 标题字数

```
最大: 80字符（B站允许长标题！）
推荐: 20-50字
关键词: 前20字包含核心关键词

B站长标题的优势:
  - 可以包含更多搜索关键词
  - 可以用【】标注分类
  - 可以同时包含主标题+副标题
```

### 标题格式

```
标准格式:
  【分类】主标题 - 副标题

示例:
  "【保姆级教程】用AI写代码从入门到实战 - Claude API完全指南"
  "【盘点】2024年最值得用的10个AI工具，第5个绝了"
  "【AI全栈开发 EP.3】搭建智能客服系统 | 从0到1实战"
```

## 简介/描述规范

### 简介区

```
字数: ≤2000字
格式: 纯文本，支持链接

推荐结构:
  第1段: 视频内容概要
  第2段: 时间轴/目录（如有）
  第3段: 相关链接/资源
  第4段: 互动引导

时间轴格式:
  00:00 开场
  01:30 第一部分: XXX
  05:20 第二部分: XXX
  09:00 总结
```

### 标签

```
数量: 最多10个标签
选择:
  - 与内容相关的关键词
  - 分区相关标签
  - 系列名称（如有）

注意:
  - 标签影响搜索和推荐
  - 不要用无关标签蹭流量
  - 一级标签 + 二级标签 + 系列标签
```

## 分P功能

```
B站独有: 多P（分集）功能
  - 一个稿件可以包含多个视频
  - 适合: 教程系列、合集、长视频分集

规范:
  - 每P都可以单独命名
  - P1 的封面 = 整个稿件的封面
  - 每P时长建议统一
  - 分P数无硬性限制
```

## 互动组件

### 投票

```
在视频中插入投票:
  - 选项: 2-4个
  - 时间: 可设置投票出现的时间点
  - 效果: 增加互动数据
```

### 弹幕池

```
弹幕参数:
  - 类型: 滚动/顶部固定/底部固定
  - 颜色: 可自定义
  - 字号: 小/中/大
  - 弹幕池容量: 根据视频时长自动分配

UP主管理:
  - 可设置弹幕关键词屏蔽
  - 可关闭弹幕
  - 可设置弹幕开放时间
```

## 音频规范

### 背景音乐

```
安全来源:
  - B站音频库
  - 已获授权音乐
  - CC0 无版权音乐
  - 原创音乐

音量建议:
  - BGM: 10-20%（B站视频通常以内容为主）
  - 人声: 80-90%
  - 音效: 5-15%

B站用户偏好:
  - 解说/教程类: BGM要小，人声清晰
  - 二创/MAD: BGM是核心
  - vlog: BGM适中
```
