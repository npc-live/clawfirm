# B站内容规则与审核

## 社区规范核心

### 禁止内容

1. **违法违规**：涉政敏感、暴力恐怖、色情低俗
2. **虚假信息**：谣言、伪科学、虚假宣传
3. **侵权行为**：版权侵犯、抄袭搬运
4. **不当言论**：人身攻击、地域歧视、引战
5. **不当营销**：未声明的商业推广、恶意引流
6. **低质量内容**：纯搬运、无意义内容、标题党

## 稿件审核标准

### 审核流程

```
提交 → 机器审核（分钟级）
  ├── ✅ 通过 → 直接发布
  ├── ⚠️ 待人审 → 排队人工审核（小时级）
  └── ❌ 不通过 → 打回 + 原因说明

发布后:
  → 定期巡检
  → 用户举报触发复审
  → 大流量触发复审
```

### 自制 vs 转载（重要）

```
自制:
  - 标记为"自制"的稿件享有更高推荐权重
  - 必须是原创内容
  - 错误标记为自制会被处罚（扣分+限流）

转载:
  - 必须标注原作者和来源
  - 不享受创作激励
  - 推荐权重低于自制
  - 转载未标注 → 视为搬运 → 删除+扣分

注意:
  - 翻译/字幕组作品: 建议标为"转载"并标注原始来源
  - 二次创作: 取决于原素材是否允许二创
  - 混剪/盘点: 需要声明素材来源
```

## 版权规则（B站特别严格）

### 音乐版权

```
安全:
  - B站音频库中的音乐
  - 已购买版权/获得授权
  - CC0 / Creative Commons 音乐
  - 原创音乐

风险:
  - 流行音乐直接使用
  - 游戏/动漫 OST（部分有限制）
  - 其他视频的BGM

后果:
  - 视频被静音
  - 视频被删除
  - 收到版权投诉（影响账号）
```

### 影视素材

```
允许（通常）:
  - 合理引用（评论/解说中引用片段）
  - 二次创作（MAD/AMV，需标注来源）
  - 预告片/官方素材

禁止:
  - 完整搬运影视内容
  - 未标注来源的片段
  - 付费内容的盗版

B站特点:
  - 对二次创作相对宽容（这是B站文化）
  - 但版权方发起投诉时必须下架
  - 建议在简介中标注素材来源
```

## 弹幕/评论区规则

### 弹幕审核

```
禁止弹幕:
  - 人身攻击/辱骂
  - 剧透（在非剧透区）
  - 广告/引流
  - 涉政敏感
  - 色情/低俗
  - 刷屏（连续发送相同内容）

UP主权限:
  - 可以关闭弹幕
  - 可以屏蔽关键词
  - 可以举报/删除弹幕
```

### 评论区审核

```
自动过滤:
  - 敏感词自动过滤
  - spam/广告自动折叠
  - 负面内容可设置审核后显示

UP主管理:
  - 可以置顶评论
  - 可以精选评论
  - 可以删除评论
  - 可以设置评论权限
```

## 恰饭（商业合作）规范

### 必须声明

```
B站对商业合作的声明要求:
  1. 视频中口头说明（推荐在开头）
  2. 使用B站"商业合作"标签
  3. 简介中标注

不声明的后果:
  - 被观众举报"恰饭不报"（B站社区非常反感这个）
  - 视频限流
  - 账号信誉下降
  - UP主口碑受损（比平台处罚更严重）
```

### 花火平台

```
B站官方商业合作平台:
  - 品牌通过花火对接UP主
  - 合作内容有官方标记
  - 数据透明

建议:
  - 有条件尽量走花火
  - 保持内容独立性（即使是商单也要做真实测评）
  - B站观众接受恰饭，但不接受虚假推荐
```

## 禁用词/敏感词

### 通用禁用词

```
政治敏感: 同其他平台
绝对化用语: 最、第一、唯一、永久等
虚假承诺: 保过、包就业、一定能、100%
医疗断言: 治愈、根治、特效药
```

### B站特殊注意

```
引战: 比较不同游戏/番剧/偶像时注意措辞
地域: 避免地域刻板印象
饭圈: 避免挑动粉丝群体对立
敏感历史: 谨慎引用历史事件
```

## 处罚机制

### 处罚等级

```
Level 1 - 警告
  - 内容被删除
  - 收到系统通知

Level 2 - 禁言
  - 禁止评论/弹幕一段时间
  - 1天/7天/30天

Level 3 - 限流
  - 视频不被推荐
  - 持续时间不定

Level 4 - 封禁
  - 临时封禁（7天/30天）
  - 永久封禁（严重违规）
```

### 申诉

```
如果认为误判:
  1. 在"创作者服务中心"提交申诉
  2. 提供申诉理由和证据
  3. 等待审核（通常1-3个工作日）
  4. 申诉结果会通过系统消息通知
```
